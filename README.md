# Chatbot pandacola

## Project setup
```
npm install
```

### Start Vue client
```
npm run serve (in /src)
```

### Start Node server
```
nodemon app.js (in /server)
```

### Start MongoDb
```
sudo mongod
```

### Display Diagram tree
```
Go to https://draw.io -> open -> Diagram Chatbot Pandacola.xml
```
